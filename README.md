## ğŸ“– Overview

This repository hosts the official curriculum and source codes for the **SPREP ClimSA 2025 Capacity Building Program**. It is designed to empower **National Meteorological & Hydrological Services (NMHSs)** with advanced IT capabilities.

The training focuses on **operational applicability**, covering the entire data lifecycle from extraction to automated reporting, with a special emphasis on integrating with existing tools like **CliDE** and **CliDEsc**.

### ğŸ¯ Key Learning Objectives
* **Data Processing**: Handling NetCDF, GRIB, and Station data using Python & R.
* **System Integration**: Connecting scripts to Climate DBs (CliDE) and external APIs.
* **Automation**: Building robust pipelines with Cron and Shell scripting.
* **Visualization**: Generating publication-ready climate maps and automated PDF reports.
